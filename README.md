Within this project, I will make a website which lets users generate a personalised workout split for users. It will take in user prefrences as inputs (such as how many days they want to train per week, how experienced they are, how long they want each workout to be) and use that information to make a training program. 
Within the website, there will also be a calorie calculator, which the can be used by the user to see how many calories they need to eat per day to progress towards their goals.
